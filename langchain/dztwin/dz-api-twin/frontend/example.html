AI Chat Widget Demo

Click the chat button in the bottom-right corner to start a conversation.

🚀 One-Line Integration

Add this single script tag to any website:

<script src="chat-plugin.js"></script>

🎨 Customizable

Configure the widget with a simple JavaScript object:

window.ChatPluginConfig = { apiUrl: 'http://localhost:3000/api', theme: 'default' };

🤖 Multiple AI Providers

Supports AWS Bedrock, OpenAI, Anthropic, Google Gemini, and Azure OpenAI.

📚 RAG Support

Enhanced with semantic search over your API documentation for accurate, grounded responses.

💬 Streaming Responses

Real-time token-by-token display for a smooth user experience.

🔒 Iframe Isolation

Zero CSS/JS conflicts with your existing website.
